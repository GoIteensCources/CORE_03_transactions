# MINI project transacctions

Метою проєкту є створення системи управління
особистими фінансами.

Уявіть, що користувач No1 хоче мати повний контроль над своїми фінансами, включаючи доходи, витрати,
бюджети та збереження. Він бажає мати можливість
легко реєструватися в системі, входити у свій
обліковий запис, додавати фінансові записи,
відслідковувати витрати та отримувати звіти про свій
фінансовий стан.
